Señal Amortiguada

En este ejemplo, vamos a explorar una señal amortiguada. Este tipo de señal se utiliza en sistemas donde es importante observar cómo las oscilaciones se reducen con el tiempo, como en sistemas mecánicos o eléctricos.
% La fórmula que define la señal es:
%
% $$ y = \sin(x) \cdot e^{-0.4x} $$
%
% donde la función seno representa las oscilaciones y el término exponencial
% representa la amortiguación.

Código para Generar la Señal

A continuación, definimos los valores de `x` y calculamos `y` usando la fórmula mencionada. Luego, trazamos el gráfico de la señal.
% Definición de los datos
x = 0:0.2:10;
y = sin(x) .* exp(-0.4 * x);
 
% Trazado de la gráfica
plot(x, y)
% grid on
xlabel('x')
ylabel('y')
title('Señal Amortiguada: y = sin(x) * exp(-0.4 * x)')

Explicación del Gráfico

En el gráfico, podemos observar que la amplitud de la señal disminuye con el tiempo, debido a la amortiguación exponencial. Este disipa gradualmente.

Aplicaciones de este Tipo de Señal

Las señales amortiguadas son muy útiles en:
- Circuitos eléctricos, donde se necesitan filtros para reducir oscilaciones no minimizar las oscilaciones en la respuesta del sistema.
