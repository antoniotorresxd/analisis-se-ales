Visualización de una Cara Simple en 2D

En este ejemplo, vamos a crear una representación gráfica de una cara simple utilizando contornos y puntos en el plano `xy`. La gráfica se compone de un contorno que representa la forma de la cara, dos ojos y una boca.

Configuración y Limpieza del Entorno

Limpiamos el espacio de trabajo y configuramos los ejes para una visualización adecuada.
clear; clf; hold off; % Limpiar variables, cerrar figuras y mantener el estado de la figura
dt = pi/20; % Definición del paso para la parametrización
t = 0:dt:2*pi; % Creación de un vector de tiempo de 0 a 2π
 
% Definición de las coordenadas para el contorno de la cara
x = cos(t); % Coordenadas x para el contorno de la cara
y = sin(t); % Coordenadas y para el contorno de la cara
axis([-1 1 -1 1]); hold on; % Configuración de los ejes y mantener la figura
plot(x, y); % Trazado del contorno de la cara

Dibujo de los Ojos

Utilizamos un bucle para dibujar los ojos, que se crean como formas elípticas que se reducen en tamaño hacia el centro de la cara.
for k = 0.8:-0.05:0.05
plot(k*0.1*x - 0.3, k*0.15*y + 0.1); % Ojo izquierdo
plot(k*0.1*x + 0.3, k*0.15*y + 0.1); % Ojo derecho
end

Dibujo de la Boca

Se define una semicircunferencia para representar la boca, que se traza en la parte inferior del contorno de la cara.
s1 = 3*pi/2 - 1.1; % Ángulo inicial para la boca
s2 = 3*pi/2 + 1.1; % Ángulo final para la boca
s = s1:dt:s2; % Vector de ángulos para la boca
xs = 0.5*cos(s); % Coordenadas x para la boca
ys = 0.5*sin(s); % Coordenadas y para la boca
plot(xs, ys); % Trazado de la boca
hold off; % Finalizar la figura

Explicación de la Gráfica

En esta gráfica, se presenta una representación simple de una cara. El contorno circular representa la forma de la cabeza, mientras que los trazos elípticos para los ojos y la curva semicircular para la boca completan la cara.
Este tipo de visualización es útil para entender conceptos básicos de gráficos en 2D y cómo se pueden combinar diferentes formas para crear representaciones más complejas.
Los elementos de la figura se escalan y se posicionan de manera intuitiva para crear una cara caricaturesca. La configuración de ejes se establece para que la proporción sea cuadrada, asegurando que las dimensiones de la cara no se distorsionen.
